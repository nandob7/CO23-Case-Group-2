you can run the validator as follows.

1. put your instance file and your solution file in the same folder as this code
2. open a terminal and in it, move to this folder
3. replace the instance and solution filenames in the following, copy it to your terminal and press enter:

       python Validate.py -i testInstance.txt -s testSolution.txt 



If this doesn't work, please check if you have python installed and if you are really in the right folder.
